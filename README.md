# Pelada-Parser

this project exist for 2 reasons:
- Learn Rust
- Detach the parsing from the [PeladaAPI](https://github.com/Horizon-XT/PeladApi) project.

##  Install

### With Zsh
1. Clone the repo.
2. Run `install.sh`

### Without Zsh
1. Clone the repo
2. Run `cargo build --release`
3. Add the binary to your path

## Usage

Start as CLI:
```bash
cargo run -- --cli <filepath>
```
